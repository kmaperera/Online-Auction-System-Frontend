# 🛒 Online Auction System (Frontend)

> A modern React-based frontend for an online auction platform that enables users to browse auctions, place bids, and manage listings through an intuitive user interface.

---

## ✨ Features

- 🔐 User Authentication
- 🛍️ Browse Auction Listings
- 💰 Place Bids
- 👤 User Dashboard
- 📱 Responsive User Interface

---

## 🛠️ Tech Stack

| Category | Technology |
|----------|------------|
| Frontend | React |
| Styling | Bootstrap |
| API Communication | Axios |
| Backend | ASP.NET Web API |

## 🚀 Getting Started

### Clone the repository

```bash
git clone <repository-url>
```

### Install dependencies

```bash
npm install
```

### Run the application

```bash
npm run dev
```
